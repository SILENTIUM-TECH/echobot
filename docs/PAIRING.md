# Pairing Your Number

Pairing links Echo to your WhatsApp the exact same way WhatsApp's own **Linked Devices** feature works — a temporary code, not a password. There's nothing to install and nothing to configure. Pick whichever of the three ways below is easiest for you.

> Echo already runs 24/7 — pairing is the entire process. You don't need a server, a computer staying on, or any technical setup.

## Method 1 — Web Dashboard

**[echowabot.vercel.app](https://echowabot.vercel.app)**

1. Open the dashboard.
2. Go to **WhatsApp Sessions**.
3. Enter your number in international format, no `+` or spaces (e.g. `256757123456`).
4. Tap **Get Pairing Code** — an 8-character code appears.
5. On your phone: **WhatsApp → Settings → Linked Devices → Link a Device → Link with phone number instead**.
6. Enter the code. You're connected.

The dashboard also lets you send messages, view live activity, and toggle group protections — all without typing a single command.

---

## Method 2 — Telegram Bot

**[t.me/devaura_echobot](https://t.me/devaura_echobot)**

1. Open a chat with the bot and press **Start**.
2. Send your WhatsApp number when asked.
3. You'll get a pairing code back.
4. On your phone: **WhatsApp → Settings → Linked Devices → Link a Device → Link with phone number instead**.
5. Enter the code. You're connected.

---

## Method 3 — Directly Through WhatsApp

If you already know someone with Echo paired, you can request a code straight from a chat with the bot:

```
.pair 256757123456
```

- Message it privately (not in a group).
- Use international format, digits only.
- You'll get a code back the same way as the other two methods.

---

## After pairing

Send `.menu` to Echo to see everything it can do. If a code expires before you enter it, just request a new one — no harm done, no limit on attempts.

## Unpairing

You're always in control — unlink Echo any time from **WhatsApp → Settings → Linked Devices**, tap the session, and remove it. No approval needed, takes effect immediately.
