# FAQ

**Do I need to install anything?**
No. Echo runs already — pairing your number is the entire process. No app, no server, no setup.

**Is it free?**
The core bot is free to pair and use. Some advanced features are part of an optional premium tier — see `.myprem` after pairing, or the Store section on the [dashboard](https://echobot.eu.org).

**Will my number get banned?**
Linking a bot to WhatsApp through methods like this isn't officially supported by WhatsApp, and heavy automated/spam-like behavior on any linked device can get a number flagged. Normal usage — chatting, using commands, group moderation — is how the vast majority of users run it without issue. If you're cautious, pair a secondary number rather than your primary one.

**Can Echo read my private messages with other people?**
No. It only processes messages sent directly to it (commands, or messages in a chat where you've enabled the AI assistant) — it isn't reading your other conversations.

**How do I remove it?**
WhatsApp → Settings → Linked Devices → find the session → remove it. Immediate, no approval needed.

**Can I run more than one number?**
Yes — pair as many as you like through the dashboard or Telegram bot; each runs independently.

**Something isn't working — where do I report it?**
[t.me/silentium_01](https://t.me/silentium_01) for announcements and support, or the community group.

**Is the "verified" look on Echo's messages an actual WhatsApp verification?**
No — it's Echo's own visual branding (styled cards, consistent formatting). It doesn't change your account's status or grant any official badge.
