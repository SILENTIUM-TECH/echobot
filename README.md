## Why people pair Echo

### 🧠 An AI that's actually *in* the conversation
Not a bolt-on chatbot — talk to Echo like you'd talk to a person. It remembers your last conversation, matches your tone, generates images on request, and pulls **real, live information** into its answers — current weather, breaking news, crypto prices, the actual date — instead of guessing. Ask it something. Watch it actually know.

### 👁️ Nothing disappears on you anymore
Someone sends a view-once photo, video, or voice note? Echo can recover it. No more "this can only be viewed once" walls.

### 🛡️ Groups that defend themselves
Set it once, forget it exists — until it saves you:
- **Anti-Link** — auto-warns, kicks, or deletes the moment someone drops a link
- **Anti-Group-Status-Mention** — stops randoms from farming your group into their WhatsApp Status
- **Anti-Delete** — catches messages people try to delete before you see them
- **Anti-Demote / Anti-Kick / Anti-Promote** — locks down admin powers so a compromised or rogue admin can't quietly take over
- **Warning System** — three strikes, auto-kick, fully logged
- **Slowmode, mute scheduling, welcome/goodbye messages** and a dozen more toggles, all one command away

### 📥 Anything you can link, Echo can pull
YouTube, TikTok, Instagram, Facebook, Twitter/X, Spotify, Pinterest — drop a link or just say the song name. Audio downloads arrive with real title/artist/cover-art tags, so they show up properly in your music player, not as some random filename.

### 📡 Broadcast like a brand, not a bot

## Is my data safe?

Short, honest answer — see the full breakdown in **[docs/PRIVACY.md](docs/PRIVACY.md)** — but the summary:

- Pairing works exactly like WhatsApp's own **Linked Devices** feature. You're not handing over a password, because there isn't one — WhatsApp itself generates the temporary code.
- Echo only processes what it needs to answer you: your command, the message you're replying to, and (for the AI) recent conversation context so it doesn't forget what you just said.
- Your chats aren't sold, scraped for ads, or handed to anyone. Group moderation features (like the warning system) only store what's needed for *that group's own admins* to moderate — not shared elsewhere.
- You can unlink at any time from WhatsApp's own **Linked Devices** screen — no need to ask permission, no waiting period.
